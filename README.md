## Installation on ubuntu

  * Create Virtual Environments
  
            $pip install --user virtualenv
            $conda create -n myenv python=3.6 (python>=3)
            $conda activate myenv

  * Pandas
            ```
            $conda install Pandas
            ```
  * Flask
            ```
            $conda install flask 
            ```
  * requests
            ```
            $conda install requests 
            ```
  * jsons
            ```
            $pip install jsons
            ```
